<h1>Project Overview</h1>
<p>The <strong>HR Attrition Analysis</strong> project is a cross-functional data analytics initiative designed to uncover why employees leave companies. Using simulated data from a fictional tech firm, we walk through the entire data pipeline:</p>
<ul>
  <li>✅ <strong>Data Cleaning</strong></li>
  <li>✅ <strong>Data Modeling</strong></li>
  <li>✅ <strong>Visualization</strong></li>
  <li>✅ <strong>Business Recommendations</strong></li>
</ul>

<ul style="font-size:18px; line-height:1.8;">
  <li><strong>Data Cleaning</strong> → Checked for missing values, incorrect types, duplicates</li>
  <li><strong>Data Modeling</strong> → Built a star schema</li>

<h1>How to Use</h1>
<div>
  <h3>• Explore Excel Files</h3>
  <p>Open <code>.xlsx</code> files for raw data and pivot tables.</p>

  <h3>• Run Python Notebooks</h3>
  <p>Ensure you have installed the required packages:</p>
  <pre><code>pip install pandas matplotlib seaborn</code></pre>

  <h3>• Open Dashboards</h3>
  <p>Power BI → <code>.pbix</code> files  <br>Tableau → <code>.twbx</code> or <code>.twb</code> files</p>

  <p>⚠ <strong>Important:</strong> Some file paths may need adjustment depending on your local setup.</p>
</div>
